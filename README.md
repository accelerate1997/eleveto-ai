# Elevetoai | AI-First Development Agency

Elevetoai is a premium, single-page web application for an AI-focused development agency. It bridges the gap between manual operations and autonomous efficiency through custom AI agents, business portals, and rapid MVP prototyping.

## ðŸ’» Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/eleveto-ai.git
   cd eleveto-ai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the local development server:
```bash
npm run dev
```
Open `http://localhost:5173` in your browser.

### Build

Generate optimized production files:
```bash
npm run build
```
The output will be in the `dist` folder.
